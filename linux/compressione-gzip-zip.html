Compressione File: gzip e zip

üìã Descrizione

Questo appunto mostra come comprimere e decomprimere file su Linux usando i due comandi principali:

  • gzip - Compressione GNU zip (standard Linux/Unix)
  • zip - Compressione compatibile Windows/cross-platform

Entrambi riducono la dimensione dei file per risparmiare spazio disco e velocizzare trasferimenti.

üì¶ gzip - Compressione GNU

gzip √® lo standard di compressione su Linux/Unix. Crea file con estensione .gz.

Sintassi Base:

Bash 
gzip file.txt

Cosa Succede:

  • File originale file.txt viene sostituito da file.txt.gz
  • Il file originale viene cancellato
  • Estensione .gz viene aggiunta automaticamente

Opzioni Principali:

-k (--keep)
Mantiene il file originale
Non cancella il file dopo compressione
gzip -k file.txt ‚Üí crea file.txt.gz e mantiene file.txt
-d (--decompress)
Decomprime
Equivalente a gunzip
gzip -d file.txt.gz ‚Üí ricrea file.txt
-1 a -9
Livello di compressione
-1 = Veloce, compressione minima
-6 = Default, bilanciato
-9 = Lento, compressione massima
Esempio: gzip -9 file.txt
-r (--recursive)
Comprime ricorsivamente
Comprime tutti i file in directory e sottodirectory
gzip -r cartella/
-v (--verbose)
Modalit√† verbose
Mostra nome file e percentuale riduzione
gzip -v file.txt
Output: file.txt: 65.2% -- replaced with file.txt.gz
-l (--list)
Info file compresso
Mostra dimensioni, ratio, nome originale
gzip -l file.txt.gz
-c (--stdout)
Output su stdout
Mantiene file originale, scrive compresso su stdout
gzip -c file.txt > file.txt.gz

Esempi Pratici gzip:

1. Compressione Base (sostituisce originale):

Bash 
gzip documento.txt
# Risultato: documento.txt.gz (documento.txt cancellato)

2. Compressione Mantenendo Originale:

Bash 
gzip -k documento.txt
# Risultato: documento.txt.gz (documento.txt mantenuto)

3. Compressione Massima:

Bash 
gzip -9 documento.txt
# Massima compressione (pi√π lento)

4. Compressione Multipla con Info:

Bash 
gzip -kv *.txt
# Output esempio:
# file1.txt:  72.1% -- created file1.txt.gz
# file2.txt:  65.8% -- created file2.txt.gz

5. Compressione Ricorsiva Directory:

Bash 
gzip -r logs/
# Comprime TUTTI i file in logs/ e sottocartelle

6. Info su File Compresso:

Bash 
gzip -l documento.txt.gz
# Output esempio:
#         compressed        uncompressed  ratio uncompressed_name
#              12345              123456  90.0% documento.txt

üìÇ gunzip - Decompressione

gunzip decomprime file .gz. √à equivalente a gzip -d.

Sintassi:

Bash 
gunzip file.txt.gz

# Equivalente a:
gzip -d file.txt.gz

Cosa Succede:

  • File file.txt.gz viene decompresso
  • Risultato: file.txt (originale ricreato)
  • File .gz viene cancellato

Esempi Decompressione:

1. Decompressione Base:

gunzip documento.txt.gz
# Risultato: documento.txt

2. Decompressione Mantenendo .gz:

gunzip -k documento.txt.gz
# Risultato: documento.txt (documento.txt.gz mantenuto)

3. Decompressione Multipla:

gunzip *.gz
# Decomprime tutti i .gz nella directory

4. Visualizza Contenuto Senza Decomprimere:

# Opzione 1: zcat
zcat documento.txt.gz

# Opzione 2: gunzip -c
gunzip -c documento.txt.gz

# Opzione 3: gzip -dc
gzip -dc documento.txt.gz

5. Cerca Dentro File Compressi:

# zgrep - grep su file .gz
zgrep "errore" logfile.txt.gz

# Equivalente a:
gunzip -c logfile.txt.gz | grep "errore"

üóúÔ∏è zip - Compressione Cross-Platform

zip crea archivi compressi compatibili con Windows/Mac/Linux. Estensione .zip.

Sintassi Base zip:

Bash 
zip archivio.zip file1.txt file2.txt file3.txt

Opzioni Principali zip:

-r (--recurse-paths)
Ricorsivo
Include directory e sottodirectory
zip -r backup.zip cartella/
-0 a -9
Livello compressione
-0 = Store (nessuna compressione, solo archivia)
-6 = Default
-9 = Massima compressione
-u (--update)
Aggiorna archivio
Aggiunge solo file nuovi o modificati
zip -u archivio.zip nuovofile.txt
-d (--delete)
Rimuove file da archivio
zip -d archivio.zip file.txt
-e (--encrypt)
Cripta con password
Richiede password interattiva
zip -e segreto.zip documento.txt
-q (--quiet)
Modalit√† silenziosa
Non mostra output
zip -q archivio.zip *.txt
-x (--exclude)
Esclude file/pattern
zip -r backup.zip . -x "*.log" "temp/*"

Esempi Pratici zip:

1. Comprime Singolo File:

Bash 
zip documento.zip documento.txt
# Crea documento.zip, mantiene documento.txt

2. Comprime Multipli File:

Bash 
zip documenti.zip file1.txt file2.txt file3.txt

# Oppure con wildcard:
zip documenti.zip *.txt

3. Comprime Directory Completa:

Bash 
zip -r backup.zip progetto/
# Include TUTTA la directory progetto/ ricorsivamente

4. Compressione Massima:

Bash 
zip -9 -r archivio.zip cartella/
# Massima compressione

5. Esclude File Specifici:

Bash 
# Esclude log e file temp
zip -r backup.zip progetto/ -x "*.log" "*.tmp" "node_modules/*"

8. Lista Contenuto senza Decomprimere:

Bash 
unzip -l archivio.zip
# Mostra lista file nell'archivio

üìÇ unzip - Decompressione ZIP

Sintassi Base:

Bash 
unzip archivio.zip

Opzioni Principali unzip:

-l (--list)
Lista contenuto
Non decomprime, solo mostra file
unzip -l archivio.zip
-d (--directory)
Directory destinazione
Estrae in directory specifica
unzip archivio.zip -d /tmp/estratti
-o (--overwrite)
Sovrascrivi senza chiedere
unzip -o archivio.zip
-n (--never-overwrite)
Non sovrascrivere mai
Salta file esistenti
unzip -n archivio.zip
-q (--quiet)
Silenzioso
No output
unzip -q archivio.zip
-j (--junk-paths)
Ignora path
Estrae tutti i file nella directory corrente (flat)
unzip -j archivio.zip

Esempi Pratici unzip:

1. Decompressione Base:

Bash 
unzip archivio.zip
# Estrae tutto nella directory corrente

2. Estrai in Directory Specifica:

Bash 
unzip backup.zip -d /home/user/ripristino/
# Crea directory se non esiste

3. Lista Contenuto:

Bash 
unzip -l archivio.zip
# Output:
# Archive:  archivio.zip
#   Length      Date    Time    Name
# ---------  ---------- -----   ----
#      1234  12-31-2024 10:30   file1.txt
#      5678  12-31-2024 10:31   file2.txt

4. Estrai Solo File Specifico:

Bash 
unzip archivio.zip documento.txt
# Estrae solo documento.txt

5. Estrai con Pattern:

Bash 
# Solo file .txt
unzip archivio.zip "*.txt"

# Solo file in una cartella
unzip archivio.zip "docs/*"

6. Testa Integrit√†:

Bash 
unzip -t archivio.zip
# Verifica integrit√† senza estrarre
