Questo appunto mostra tre metodi principali per visualizzare solo le cartelle (directory) escludendo i file normali:

  • ls -d */ - Metodo rapido con wildcard
  • ls -l | grep "^d" - Filtraggio con grep sui permessi
  • find - Ricerca ricorsiva avanzata

Il metodo piÃ¹ veloce e diretto per listare solo le cartelle.

Variante: Path Specifico

Bash 
ls -d /percorso/cartella/*/

Lista solo le cartelle dentro /percorso/cartella/ senza entrare ricorsivamente.

Esempio Pratico:

# Lista cartelle in /var/log/
ls -d /var/log/*/

# Output esempio:
/var/log/apache2/
/var/log/apt/
/var/log/journal/
/var/log/nginx/

Includere Cartelle Nascoste

Le cartelle nascoste (che iniziano con .) richiedono l'opzione -a.

Metodo 1 con nascoste:

Bash 
ls -la | grep "^d"

-a include file e cartelle nascoste (iniziano con .)

Alternativa pulita (esclude . e ..):

Bash 
ls -lA | grep "^d"

-A mostra nascoste ma ESCLUDE . e ..
