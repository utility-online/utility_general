Lista Tabelle di una Libreria

üìÅ Categoria: PROC SQL üìÖ Aggiornato: 12 Dicembre 2024 ‚è±Ô∏è Lettura: 6 min

üìã Descrizione

Questo appunto mostra come ottenere la lista di tutte le tabelle presenti in una libreria SAS utilizzando le viste DICTIONARY tramite PROC SQL.

√à particolarmente utile per esplorare librerie, verificare l'esistenza di tabelle, o creare report sui metadati.

üíª Codice Base

SAS 
PROC SQL;
   CREATE TABLE lista_tabelle AS 
   SELECT memname, memtype 
   FROM dictionary.tables 
   WHERE libname = 'NOME_LIBRERIA'; /* Il nome deve essere in MAIUSCOLO */
QUIT;

üìä Output Esempio

Esempio 1: Libreria SASHELP

SAS 
PROC SQL;
   SELECT memname, memtype, nobs, nvar
   FROM dictionary.tables 
   WHERE libname = 'SASHELP'
   ORDER BY memname;
QUIT;

Output:

MEMNAME MEMTYPE NOBS NVAR
CARS DATA 428 15
CLASS DATA 19 5
SHOES DATA 395 7

üîÑ Alternativa: Viste SASHELP

Le viste DICTIONARY sono disponibili solo in PROC SQL. Per usarle in DATA step, SAS fornisce viste SASHELP equivalenti:

DICTIONARY SASHELP Equivalente
DICTIONARY.TABLES SASHELP.VTABLE
DICTIONARY.COLUMNS SASHELP.VCOLUMN
DICTIONARY.MEMBERS SASHELP.VMEMBER
DICTIONARY.LIBNAMES SASHELP.VLIBNAM

Esempio con SASHELP.VTABLE:

SAS 
/* In DATA step */
DATA lista_tabelle;
    SET sashelp.vtable;
    WHERE libname = 'WORK';
    KEEP libname memname nobs nvar;
RUN;
