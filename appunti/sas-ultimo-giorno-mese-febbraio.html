Calcolare l'Ultimo Giorno del Mese

Questo appunto mostra come calcolare l'ultimo giorno di qualsiasi mese, incluso febbraio con la gestione degli anni bisestili.

Il codice utilizza CALL SYMPUT per creare una macro variabile contenente l'ultimo giorno del mese specificato.

SAS 
IF &MESEFINE IN(1,3,5,7,8,10,12) THEN 
    CALL SYMPUT("GGFINE", 31);

IF &MESEFINE IN(4,6,9,11) THEN 
    CALL SYMPUT("GGFINE", 30);

IF &MESEFINE IN(2) THEN 
    CALL SYMPUT("GGFINE", 28 + (MOD(&ANNOFINE, 4) = 0));
