Opzioni SAS - Debug e Performance

Le opzioni SAS permettono di controllare il comportamento del sistema durante l'esecuzione dei programmi. Questo appunto raccoglie le principali opzioni per il debug delle macro e il monitoraggio delle performance.

Opzioni Debug Macro

Attivare Debug:

SAS 
OPTIONS MPRINT SYMBOLGEN MLOGIC;
MPRINT
Macro Print
Stampa nel log il codice SAS generato dalle macro dopo la risoluzione.
Mostra esattamente quale codice viene eseguito.
SYMBOLGEN
Symbol Generation
Mostra nel log il valore di ogni macro variabile quando viene risolta.
Utile per vedere cosa contengono le variabili &var.
MLOGIC
Macro Logic
Traccia l'esecuzione della logica delle macro (%IF, %DO, ecc.).
Mostra quali rami del codice vengono eseguiti.

Disattivare Debug:

SAS 
OPTIONS NOMPRINT NOSYMBOLGEN;

Prefisso NO disattiva l'opzione (es: NOMPRINT, NOSYMBOLGEN, NOMLOGIC).

Opzione Compilazione Macro

SAS 
OPTIONS MCOMPILENOTE=ALL;
MCOMPILENOTE=ALL
Macro Compile Note
Scrive note nel log sulla compilazione delle macro.
Mostra quando una macro viene compilata e memorizzata.
Valori: ALL, NONE, NOAUTOCALL

Opzione Livello Messaggi

SAS 
OPTIONS MSGLEVEL=I;
MSGLEVEL=I
Message Level - Informative
Controlla il livello di dettaglio dei messaggi nel log.
I (Informative) = Mostra messaggi informativi aggiuntivi
N (Normal) = Solo messaggi standard
Utile per ottenere pi√π informazioni su merge, indicizzazione, ordinamento.

Opzioni Monitoraggio Performance

SAS 
OPTIONS STIMER MEMRPT FULLSTIMER STATS;
STIMER
Statistics Timer
Scrive statistiche di timing nel log per ogni step.
Mostra tempo CPU e tempo reale di esecuzione.
MEMRPT
Memory Report
Riporta l'utilizzo di memoria per ogni step.
Mostra memoria allocata e memoria effettivamente utilizzata.
FULLSTIMER
Full Statistics Timer
Versione estesa di STIMER con statistiche pi√π dettagliate.
Include I/O, page faults e altre metriche di sistema.
STATS
Statistics
Scrive statistiche aggiuntive su operazioni di I/O.
Mostra numero di record letti/scritti e buffer utilizzati.
