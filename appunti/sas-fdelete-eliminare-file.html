Eliminare File con FDELETE

üìÅ Categoria: Data Step üìÖ Aggiornato: 10 Dicembre 2024 ‚è±Ô∏è Lettura: 5 min

üìã Descrizione

Questo appunto mostra come eliminare un file dal file system utilizzando le funzioni SAS FILENAME, FEXIST e FDELETE.

Il codice verifica prima se il file esiste e lo elimina solo se presente, evitando errori.

Codice

SAS 
DATA _null_;
    fname = 'tempfile';
    rc = FILENAME('fname', "&indicatori./stg/mysas.sas");
    
    IF rc = 0 AND FEXIST(fname) THEN
        rc = FDELETE(fname);
        
    rc = FILENAME(fname);
RUN;

Metodo Alternativo

Usando X Statement (solo Windows/Linux):

SAS 
/* Windows */
X "del &indicatori.\stg\mysas.sas";

/* Linux/Unix */
X "rm &indicatori./stg/mysas.sas";
